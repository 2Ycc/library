# 图书管理系统

## 技术栈

SpringBoot2 + Vue2 + ElementUI + Axios + Hutool + Mysql +
Spring Boot定时任务

本课程设计基于B站up主：程序员青戈 的开源项目pure-design https://gitee.com/xqnode/pure-design.git

~~~ 
练手项目，后续不再更新！！！
~~~
